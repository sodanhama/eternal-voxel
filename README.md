# Eternal Voxel

## Description 
This made was created in Godot to help me learn the basics, and features a simple voxel generation.

I used simple noise from Godot to generate a chunk and I added basic player and camera functions, allowing users to move around.

Play the game here: https://sodanhama.itch.io/eternal-voxel
